### Usage
```
./cache_latency [L1(D) cache size (KB)] [L2 cache size (KB)] [L3 cache size (KB)]
```

### Performance

|   Mem/Cache level   | Latency(ns)|
|---------------------|------------|
| Main memory latency |  19.631031 |
|  L1 cache latency   |  7.632923  |
|  L2 cache latency   |  11.526963 |
|  L3 cache latency   |  15.453169 |
